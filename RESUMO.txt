🚀 COMEÇAR A USAR
═══════════════════════════════════════════════════

OPÇÃO 1: Página de Boas-vindas
  1. Abra: COMECE_AQUI.html
  2. Clique em "Abrir Quiz"
  3. Crie conta ou use demo (demo/1234)

OPÇÃO 2: Direto na Aplicação
  1. Abra: index.html
  2. Crie nova conta ou faça login
  3. Comece a responder quizzes

OPÇÃO 3: Com Dados de Exemplo
  1. Abra: index.html
  2. Pressione F12 (DevTools)
  3. Vá para Console
  4. Cole código de: DADOS_EXEMPLO.js
  5. Faça login com demo/1234
  6. Veja histórico pré-carregado

🆘 TROUBLESHOOTING RÁPIDO
═══════════════════════════════════════════════════

Problema: Imagens não aparecem
Solução: Verifique se 1.png até 6.png estão na pasta

Problema: Dados não salvam
Solução: localStorage pode estar desabilitado no navegador

Problema: Página fica em branco
Solução: Abra DevTools (F12) e veja console para erros

Problema: Login não funciona
Solução: Crie conta antes; senhas são case-sensitive

Problema: Quiz não avança
Solução: Selecione uma resposta antes; recarregue página
